


<h1>Wanderlust - Full Stack Airbnb Clone</h1>
    <p>Wanderlust is a comprehensive vacation rental marketplace that allows users to explore, list, and review unique accommodations. Inspired by Airbnb, this platform focuses on seamless user experience, dynamic pricing displays, and a robust community feedback system.</p>

  <hr>

  <h2>Key Features</h2>
    <ul>
        <li><strong>Exploratory Homepage:</strong> Browse a wide variety of destinations. Includes a <strong>Tax Toggle</strong> to switch between "Price per night" and "Total price including taxes."</li>
        <li><strong>Listing & Detail Views:</strong> Dedicated pages for every property featuring images, descriptions, and location details.</li>
        <li><strong>User Reviews & Ratings:</strong> Integrated feedback system where users can leave ratings and comments, and view prior community feedback.</li>
        <li><strong>Become a Host (CRUD):</strong> Authenticated users can create and manage their own listings. Once submitted, listings appear instantly on the homepage.</li>
        <li><strong>Secure Authentication:</strong> Efficient Signup and Login flow to protect user data and personalize the experience.</li>
    </ul>

  <h2> Tech Stack</h2>
    <p>
        <li>Frontend:</li>HTML5, CSS3, JavaScript (EJS Templates)<br>
        <li>Backend:</li> Node.js, Express.js<br>
        <li>Database:</li> MongoDB (Mongoose)<br>
        <li>Auth:</li>Passport.js
    </p>
